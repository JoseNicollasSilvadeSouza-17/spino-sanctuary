O projeto **SpinoSanctuary** é o Santuário de Espinossauros feito na linguagem de programação **Move** da blockchain **Sui** para o Bootcamp Sui Move Brasil realizado em Novembro (03 a 17). 

A motivação para o projeto surgiu da minha paixão por Espinossauros e pela temática de zoológicos e santuários. Achei a ideia de criar um santuário no mundo Web3 fascinante, onde o usuário pode escolher sua versão favorita de espinossauro (ou parentes próximos) e cuidar dele virtualmente.
